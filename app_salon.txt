=========================================================
  APLIKASI BOOKING SALON - PANDUAN CEPAT
=========================================================

ISI FOLDER INI:
- src/salonapp/             -> 5 file kode Java (Koneksi, FormLogin, FormUtama, FormBooking, Main)
- database_salon.sql        -> script untuk membuat database
- lib/                      -> taruh file driver MySQL (mysql-connector-j-x.x.x.jar) di sini
- nbproject/                -> file konfigurasi project NetBeans (jangan diubah-ubah)

LANGKAH SINGKAT:

1. Nyalakan Apache & MySQL di XAMPP Control Panel.
2. Buka http://localhost/phpmyadmin, klik tab SQL, paste isi file
   database_salon.sql, klik Go.
3. Download driver MySQL Connector/J (file .jar) dari:
   https://dev.mysql.com/downloads/connector/j/
   Pilih "Platform Independent", lalu download file .zip,
   ekstrak, cari file bernama mysql-connector-j-x.x.x.jar
   -> Taruh file jar tersebut ke dalam folder lib/ pada project ini.
4. Buka NetBeans -> File -> Open Project -> pilih folder SalonApp ini.
5. Klik kanan project SalonApp -> Properties -> Libraries ->
   Add JAR/Folder -> pilih file jar di folder lib/ -> OK.
6. Klik kanan project -> Clean and Build.
7. Tekan F6 (atau klik kanan project -> Run) untuk menjalankan aplikasi.
8. Login dengan:
   Username : admin
   Password : admin123

CARA PAKAI APLIKASI:
1. Setelah login, klik "FORM BOOKING / RESERVASI SALON".
2. Isi nama pelanggan, pilih layanan, isi tanggal (format: yyyy-MM-dd,
   contoh: 2026-07-15), pilih status, lalu klik Simpan.
3. Klik salah satu baris di tabel untuk mengisi ulang form (lalu bisa
   Ubah atau Hapus data tersebut).

Jika ada error, cek bagian Troubleshooting di bawah ini:

- "Driver tidak ditemukan": pastikan jar MySQL Connector/J sudah
  ditambahkan ke Libraries project (lihat langkah 5).
- "Gagal terhubung ke database": pastikan MySQL di XAMPP sudah running
  dan database db_salon sudah dibuat (lihat langkah 2).
- ComboBox Layanan kosong: pastikan tabel tb_layanan sudah memiliki
  data (cek lewat phpMyAdmin).
